# Web Monetization Flows

## Prerequisites

- Node (recommended version: LTS)
- Node package manager (npm, yarn, pnpm)

## Running the application

1. From the repository root, install the dependencies using your package manager:
```sh
# pnpm
pnpm install

# npm
npm install

# yarn
yarn install
```

2. Start the server
```sh
# pnpm
pnpm serve 

# npm
npm run serve

# yarn
yarn run serve 
```
